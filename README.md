# 🛒 Olist E-Commerce — Data Analysis

A complete Exploratory Data Analysis of 100,000+ real orders from Olist, Brazil's largest e-commerce platform.

---

## 📊 Project Overview

This project covers end-to-end data analysis including data cleaning, feature engineering, and visualizations to extract meaningful business insights from the Olist dataset.

---

## 🔍 What I Did

- Loaded and explored 7 different datasets
- Handled missing values, duplicates, and wrong data types
- Converted date columns and fixed text inconsistencies
- Created new features like delivery days and late delivery flag
- Built 7 professional visualizations

---

## 📈 Key Findings

- 96% orders were successfully delivered
- Average delivery time is 12 days
- Top selling category is Bed, Bath & Table
- Credit Card is the most preferred payment method (74%)
- Average customer review score is 4.09 out of 5
- Revenue grew consistently from 2017 to 2018
- 7.9% orders were delivered late
